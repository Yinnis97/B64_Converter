# PNG to Base64 converter


## What is Base64   
Base64 is a binary-to-text encoding that uses 64 printable characters to represent each 6-bit segment of a sequence of byte[1] values.   
As for all binary-to-text encodings, Base64 encoding enables transmitting binary data on a communication channel that only supports text.   


## Application

### 1. How to build
```cmd
make
png_reader.exe
```
### 2. How to use
Only supports PNG's for now.

Enter the path of the image :
